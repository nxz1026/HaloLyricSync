# USB 通信协议说明

## 概述

本文档说明 HALO OIXELBAR 歌词同步软件与音箱之间的 USB 通信协议。

## 通信参数

- **波特率**: 9600 bps (可配置)
- **数据位**: 8
- **停止位**: 1
- **校验位**: 无

## 协议格式

### 基本帧结构

```
[前导码 2字节][数据长度 2字节][数据...][校验和 1字节]
```

- **前导码**: `0xAA 0x55`
- **数据长度**: 小端序 (Little Endian)
- **数据**: UTF-8 编码的文本数据
- **校验和**: 所有数据字节的异或或累加和 (示例使用累加和)

## 命令说明

### 1. 显示歌词

```
[LYRIC:INDEX:TOTAL]TEXT
```

示例:
```
[LYRIC:0:20]这是第一句歌词
```

参数说明:
- `INDEX`: 当前歌词索引 (0-based)
- `TOTAL`: 歌词总行数
- `TEXT`: 歌词文本 (UTF-8编码)

### 2. 显示歌曲信息

```
[SONG]SONG_NAME|ARTIST
```

示例:
```
[SONG]海阔天空|Beyond
```

### 3. 清空显示

```
[CLEAR]
```

## 响应格式

设备在接收到命令后，应返回响应:

```
[OK]
```

或

```
[ERROR]错误信息
```

## 注意事项

1. 每行歌词建议不超过 20 个汉字
2. 建议使用 UTF-8 编码传输
3. 超时时间建议设置为 2 秒
4. 可通过 `--baud-rate` 参数调整波特率

## 协议可定制

本协议为示例协议，用户可根据实际设备的通信协议进行修改。
请查看 `usb_comm.py` 文件中的 `send_text` 方法进行定制。
