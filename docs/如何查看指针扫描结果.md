# 查看指针扫描结果的步骤

## 📋 方法 1：在 Cheat Engine 中查看（推荐）

### 步骤 1：打开 Cheat Engine
1. 启动 Cheat Engine
2. 点击 "Select a process" 按钮
3. 选择 `cloudmusic.exe`（网易云音乐）
4. 点击 "Open"

### 步骤 2：打开 Pointer Scanner
1. 点击菜单栏 `Tools` → `Pointer Scanner`

### 步骤 3：加载扫描结果
1. 在 Pointer Scanner 窗口中，点击 `File` → `Open pointer scan results`
2. 选择文件：`C:\Users\ND\Desktop\333\111.PTR`
3. 等待加载完成

### 步骤 4：查看结果
- 您会看到一个列表，显示所有找到的指针链
- 格式类似：
  ```
  "cloudmusic.exe"+01DF44D0 -> 120 -> 8 -> 0
  "cloudmusic.exe"+01DEB4D0 -> E0 -> 8 -> E8 -> 38 -> 118 -> 8 -> 0
  ```

### 步骤 5：选择指针链
- **选择最短的指针链**（偏移数量最少的）
- 例如：`"cloudmusic.exe"+00XXXXXX -> 120 -> 8 -> 0`

### 步骤 6：验证指针（重要！）
1. **重启网易云音乐**
2. **重新播放歌曲**
3. **用 Cheat Engine 找到新的歌词地址**
4. 在 Pointer Scanner 窗口右键 → `Rescan memory`
5. 输入新的歌词地址
6. 只有重启后仍然有效的指针会保留
7. **重复步骤 1-6 至少 2-3 次**

### 步骤 7：告诉我结果
找到稳定的指针链后，告诉我格式：
```
cloudmusic.exe+偏移1 -> 偏移2 -> 偏移3 -> ... -> 歌词
```

##  常见问题

**Q: 列表太长怎么办？**
A: 先选一个最短的指针链测试，如果重启后失效，再换另一个

**Q: 为什么要重启网易云音乐？**
A: 因为歌词地址是动态分配的，每次启动都会变化。只有静态偏移（相对于模块基址的偏移）才是稳定的。

**Q: 需要验证几次？**
A: 至少 2-3 次，确保指针链每次都有效
